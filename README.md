🌱 EcoAdventure — Quiz Gamificado de Educação Ambiental






📌 Sobre o projeto

EcoAdventure é um quiz gamificado de educação ambiental que ensina conceitos importantes sobre sustentabilidade de forma interativa, simples e acessível.

O projeto foi pensado para ser mais do que um quiz comum — ele busca promover consciência ambiental enquanto garante uma experiência inclusiva para todos os usuários.

♿ Acessibilidade e Inclusão

Este projeto foi desenvolvido com foco em acessibilidade digital:

🧑‍🦯 Compatível com leitores de tela (ARIA + feedback textual)
👂 Interface totalmente utilizável sem áudio (suporte a pessoas surdas)
⚡ Design com baixo estímulo visual (evita piscadas e animações agressivas)
🌈 Cores pensadas para reduzir impacto em pessoas com fotossensibilidade
⌨️ Navegação por teclado
🎮 Funcionalidades
Sistema de quiz com múltiplas perguntas
Feedback imediato de respostas
Sistema de pontuação
🏆 Recorde salvo no navegador (LocalStorage)
Tela final com resultado
Reinício de partida
Persistência de progresso

O projeto aborda temas como:

🚀 Como executar o projeto
Clone o repositório:
git clone https://github.com/MarcosBarradas/EcoAdventure-
Acesse a pasta:
cd ecoadventure
Abra o arquivo index.html no navegador
📁 Estrutura do projeto
ecoadventure/
│
├── index.html
├── style.css
├── script.js
└── README.md
🏆 Sistema de recorde

O jogo salva automaticamente o maior score já alcançado no navegador usando LocalStorage.
